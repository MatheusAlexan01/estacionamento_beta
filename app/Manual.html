Manual do Usuário

Objetivo

O objetivo deste projeto é criar um sistema de gerenciamento de vagas de estacionamento utilizando HTML, CSS e JavaScript. Este sistema permite o cadastro de veículos, o monitoramento das vagas disponíveis e ocupadas, a realização de checkouts (pagamentos) e a geração de registros diários dos pagamentos realizados.

Como Utilizar

Cadastro de Veículos:

  • Acesse a página inicial do sistema.
  • Preencha todos os campos do formulário de cadastro de veículos.
  • Clique no botão "Cadastrar" para adicionar o veículo.

Checkout (Pagamento):

  • Acesse a página de Checkout.
  • Digite a placa do veículo no campo de busca e clique no botão "Buscar".
  • Insira o horário de saída e selecione o status de pagamento.
  • Clique no botão "Calcular" para calcular o valor a pagar.
  • Clique no botão "Gerar Comprovante" para gerar o comprovante do pagamento.
  • Clique no botão "Enviar para o Registro Geral" para enviar os dados para o registro diário.

Registro Diário:

  • Acesse a página de Registro Diário.
  • Visualize os registros de pagamentos na tabela.
  • Clique no botão "Limpar Tabela" para apagar todos os registros.
  • Clique no botão "Gerar PDF" para gerar um PDF com os registros diários.
